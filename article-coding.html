JavaScript Grafik-Übung

Schreibe JavaScript-Code und lasse ein Bild direkt im Browser entstehen. Der Canvas zeigt sofort das Ergebnis deines Codes.

In diesem Artikel kannst du mit JavaScript zeichnen: Erstelle Formen, Farben oder ganze Szenen. Wenn du auf "Code ausführen" klickst, wird dein Bild auf dem Canvas sichtbar.

Die Grafik entsteht komplett durch JavaScript, ohne Bilder von außen. Viel Spaß beim Programmieren!

Grundlagen: Was ist ein Kreis?

Ein Kreis hat einen Mittelpunkt und einen Radius. Auf dem Canvas zeichnest du ihn mit ctx.arc(x, y, radius, 0, Math.PI * 2). Dabei ist x die horizontale Lage, y die vertikale Lage und radius die Größe.

Programmier-Grundlagen in JavaScript:

  • const für feste Werte, z. B. const radius = 50;
  • ctx ist der Zeichenkontext für den Canvas
  • Funktionen werden mit function definiert und beim Klick ausgeführt
  • Math.PI hilft dir, ganze Kreise (360°) zu zeichnen

Wenn du strangecircle, circle oder kreis in den Editor schreibst und dann auf „Code ausführen“ klickst, erscheint automatisch ein Bild mit Kreisen.

JavaScript Code

Nutze die Variable ctx für den Canvas-Kontext und canvas für die Leinwandgröße. Schreibe strangecircle, circle oder kreis, damit ein Bild automatisch erscheint.
